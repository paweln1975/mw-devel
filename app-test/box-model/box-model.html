Wszystkim elementom nadajemy border-box

Box model zawiera:

  • content
  • padding - odległość contentu od border - powiększa element (content box), pomniejsza element (border box)
  • border - powiększa element (content box), pomniejsza element (border box)
  • margin - odległość poza elementem - nie wpływa na rozmiar elementu
